# Employee Management System

## Project Description

This project is an Employee Management System developed using HTML, Bootstrap, JavaScript, and JsonPowerDB. The application allows users to manage employee records efficiently by performing Create, Read, and Update operations on employee data stored in JsonPowerDB.

The Employee ID acts as a Primary Key, ensuring that each employee record is unique.

---

## Benefits of using JsonPowerDB

* Simple and lightweight database solution
* Schema-free JSON-based database
* High performance and low latency
* Easy REST API integration
* Serverless architecture
* Fast development and deployment

* Employee ID
* Employee Name
* Salary
* HRA
* DA
* Deduction
